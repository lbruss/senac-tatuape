Protocolos


ICMP - PING - Testa a internet
HTTP - Protocolo de internet - Protocolo não seguro
HTTPS - Protocolo de internet - Protocolo de internet seguro
TCP - Protocolo de internet que vai e vem - se encarrega de entregar o pacote, garante que o site vai abrir por completo
UDP - Protocolo de internet - Pacotes que acontece em tempo real, entrega de forma rápida, trafega rápido ou quando o IP é feito de qualquer jeito
SMTP - protocolo que envia o e-mail
IMAP - Protocolo que recebe o e-mail
DHCP - Entrega o IP automático para os computadores/maquinas
DNS - Protocolo que converte IP para nome ou inverso
TFTP - Protocolo de transferência de arquivos 
NTP - Protocolo que sincroniza o horário nas maquinas/computadores

IP fixo - voce configura manualmente, numero por numero
IP dinamico - configura automaticamente
Conflito de IP - quando o IP não entende o protocolo mandado
HTTP/HTTPS - PORTA 80 - TCP
SMTP/IMAP/POP3 - PORTA 143 - TCP
TFTP - PORTA 69 - TCP
DNS - PORTA 53 - UDP/TCP (Depende do servidor)
SSH - PORTA 22 - TCP
